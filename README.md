# 🏺 Pyramid Labyrinth

**Pyramid Labyrinth** is an atmospheric browser-based maze adventure set inside an ancient Egyptian pyramid.
Navigate a procedurally generated labyrinth, uncover hidden passages, answer knowledge-based questions, and collect fragments to unlock the final seal and escape.

---

## 🎮 Gameplay Overview

You awaken deep inside a shifting pyramid.

* The maze changes every time you play
* Doors are scattered throughout the labyrinth
* Each door challenges you with a question
* Correct answers grant **fragments (clues)**
* Wrong answers cost **health (HP)**

Your goal:

> 🧩 Collect clues → Decode the final word → Escape the pyramid

---

## 🧠 Features

### 🔀 Procedural Maze Generation

* Every run is unique
* No fixed path — true exploration required
* Doors are randomly placed

### 🤖 AI-Generated Questions

* Questions are dynamically generated
* Includes:

  * Knowledge-based questions
  * Trick questions
  * Logic-based questions
  * Scenario-style challenges

### 🧩 Final Word Puzzle

* Clues collected from doors form a hidden pattern
* Cryptic hints guide you toward the answer
* Solve the final word to win

### 🌟 Enhanced Visuals

* Improved lighting for better visibility
* Atmospheric torch-style glow
* Minimal UI for immersion

---

## 🎯 Controls

| Key               | Action              |
| ----------------- | ------------------- |
| **W / A / S / D** | Move                |
| **G**             | Interact with doors |

---

## ❤️ Health System

* You start with **50 HP**
* Wrong answers reduce HP
* If HP reaches **0 → Game Over**

---

## 🏁 Winning the Game

To escape:

1. Explore the maze
2. Find and open doors
3. Answer questions correctly
4. Collect clue fragments
5. Reach the final chamber
6. Enter the correct final word

---

## 🛠️ Installation & Running

No setup required.

1. Download the HTML file
2. Open it in your browser
3. Play instantly
